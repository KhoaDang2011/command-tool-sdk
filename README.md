# Command Tool SDK

Một command-line tool nhỏ và vui để thực hiện các tác vụ với quyền admin.

## Tính năng

- 🎮 Các lệnh vui và hữu ích
- 🔐 Quyền admin để thực hiện tác vụ hệ thống
- 🚀 Dễ sử dụng
- 📦 Có thể mở rộng

## Cài đặt

```bash
git clone https://github.com/KhoaDang2011/command-tool-sdk.git
cd command-tool-sdk
pip install -r requirements.txt
```

## Sử dụng

```bash
python main.py --help
python main.py status
python main.py system-info
python main.py list-processes
```

## Yêu cầu

- Python 3.7+
- Administrator/Root privileges
